## Installation

```bash
go get github.com/cohesity/cohesity-appspec/tools/appspecvalidator
```

## Build

```bash
cd $GOPATH/cohesity/cohesity-appspec/tools/appspecvalidator
go build appspecvalidator_exec.go
```

## Run

```bash
./appspecvalidator_exec /path/to/appSpec.yaml
```
